# 常用命令

## 常用Hexo命令

```bash
hexo init 		#初始化本地文件夹

hexo g 				#生成静态网页
hexo d				#上传github
hexo s				#本地预览

hexo clean 		#清除本地生成的页面

hexo new post "newarticlename" #新建一篇文章

hexo new page "newpagename"   #新建一个页面
```

## 常用npm命令

```bash
npm install <pk>

#卸载插件  注意pk要去除@后的内容
npm uninstall <pk>
npm uninstall --save <pk>

npm list --depth=0 #列出npm安装的包
```
